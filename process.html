Process

Here is the process in which we try to follow.

  1. Discovery
    • Learning business objectives
    • Understanding the main goal
    • Prioritize features
    • View past statistics
    • User Research
    • User Stories and Personas
    • Brainstorming session
  2. Planning
    • Information architecture
    • Build a site-map
    • Discuss navigation
    • Project milestones
  3. Wireframing
    • Basic sketches
    • Prototype
    • Test layout options
    • Construct visual hierarchy
  4. Design
    • Color, typography, images, & copy integrated
  5. Frontend Developemnt
    • HTML, CSS, & JS
  6. Backend Developemnt
    • Integrate CMS
  7. Test
    • Deploy to beta
    • Usability
    • Load
    • Accessibility
    • Train users
  8. Deploy
    • Deploy to production
  9. Review
    • Establish a road-map for improvements
    • Analytics
  10. Manage
    • Execute improvements
